Commands for TypeScript Project:

  • tsc --init ----- to create a tsconfig.json file
  • tsc --watch OR tsc -w ------- This is the flag to run the project in watch mode.

    • tsconfig.json file understanding:

  • "target": "es2016", ------ This tells you which version of js it will convert to.
  • lib:[] --------- library

    • umcommit

  • "rootDir": "./src",
  • "outDir": "./dist",
  • "sourceMap": true,

    • Note: Place your ts files in src folder and js file in dist folder
